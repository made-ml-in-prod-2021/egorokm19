Homeworks №1 for ML in production

Данные для проекта (необходимо положить данные в папку: ml_project/data/raw): [датасет](https://www.kaggle.com/ronitf/heart-disease-uci)

* Устанавливаем проект командой: ```pip install ml_project```

* Устанавливаем виртуальную среду и необходимые библиотеки:
```python -m venv .venv```
```source .venv/bin/activate```
```pip install -r requirements.txt```

* Команда, чтобы запустить EDA (exploratory data analysis) по датасету: ```python ml_project/reports/eda_reports.py``` на выходе получается файл reports.txt с полным отчетом.

* Команда, чтобы запустить проект (предобработка, обучение, валидация) ```python ml_project/train_pipeline.py ml_project/configs/train_config.yaml```

* Команда, чтобы запустить тесты для проекта ```python -m pytest ml_project/tests/```

Полный pipeline модели с чтением, трансформацией данных, обучением и предсказанием находится в файле ```train_pipeline.py```. Сначала данные считываются, разбиваются на train и valid, затем происходит заполнение пропущенных данных и применение стандартизации к данным. Обучение происходит на основе RandomForestClassifier и используется 2 конфига для train и valid. Предсказание модели оцениваются по метрикам: accuracy, f1 и precision.
В jupyter notebooks есть небольшой файл с EDA (exploratory data analysis) для ознакомления с данными.

Данный проект оцениваю на 36 баллов.
